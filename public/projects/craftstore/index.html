About

CraftStore is project made for educational purposes. I use this website mainly as a playground to practice some technologies. I try to keep this website easily scalable.

Technologies

In this project I use technologies which I want to practice. Thanks to this project I learnt how to manage global state with redux, how to set up fast and easy authentication, database with firebase. I've learnt how to set up connection with stripe through the backend express-js server to proccess payment for customers.
I try to keep this website open to adapt new technologies to practice them before I will use it in real project.

What's next?

Another thing I am about to practice on this website are tests using JEST.
Next thing I will practice is setting up whole backend logic using Django.
Website is hosted on heroku.
